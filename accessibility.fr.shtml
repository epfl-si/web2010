Accessibilité

Pour les personnes handicapées

La question de l'accessibilité des sites pour les personnes handicapées figure parmi nos priorités. Cette page est consacrée à l'accessibilité du site web de l'EPFL.

Raccourcis clavier

Toutes les pages du site de l'EPFL sont munies de raccourcis clavier (accesskeys) qui facilitent la navigation aux personnes handicapées utilisant des moyens auxiliaires tels que les logiciels de lecture d'écran, ligne braille, agrandisseur d'écran, etc.

Liste des raccourcis clavier

Les raccourcis suivants sont systématiquement placés au début de chaque page. Ils sont invisibles pour le grand public.

  • 0. Accès spécifique pour personnes handicapées
  • 1. Page d'accueil de l'EPFL
  • 2. Navigation générale pour l'ensemble de l'EPFL
  • 3. Navigation dans ce sous-site EPFL
  • 4. Accès au champ de recherche
  • 5. Début du contenu de la page
  • 6.[Non utilisé]
  • 7.[Non utilisé]
  • 8. Plan de ce sous-site
  • 9. Contact

Rappel des raccourcis clavier

Les raccourcis clavier sont différents selon le système d'exploitation et le navigateur que vous utilisez :

Pour Windows

  • Internet Explorer : utiliser ALT et la touche de raccourcis, puis Entrée
  • Firefox : utiliser ALT et la touche de raccourcis
  • Opera : utiliser ESC + MAJ et la touche de raccourcis
  • Safari : utiliser Alt et la touche de raccourcis

Pour Mac

  • Safari : utiliser CTRL, Option, puis la touche de raccourcis
  • Firefox : utiliser CTRL et la touche de raccourcis
  • Opéra : utiliser ESC + MAJ et la touche de raccourcis

Pour Linux

  • Firefox : utiliser ALT et la touche de raccourcis
  • Opera : utiliser ESC + MAJ et la touche de raccourcis

Limites des raccourcis clavier

Les touches d'accès clavier sont à priori un mécanisme d'accessibilité très séduisant par sa simplicité de mise en oeuvre. Mais les nombreux obstacles rencontrés du fait d'une implémentation inachevée en limitent considérablement la portée. Seules une poignée de touches numériques s'avèrent finalement pertinentes, et ce uniquement en renfort d'autres mesures d'accessibilité plus éprouvées.

Autres formes d'accessibilité Web

Pour aider les personnes malvoyantes, les raccourcis clavier zoom et dézoom sont utilisables.

  • Pour faire un zoom sur Windows : utiliser CTRL puis la touche +
  • Pour faire un zoom sur Mac : utiliser CMD (pomme) puis la touche +
  • Pour faire un zoom sur Linux : utiliser CTRL puis la touche +
  • Pour faire un dézoom sur Windows : utiliser CTRL puis la touche -
  • Pour faire un dézoom sur Mac : utiliser CMD (pomme) puis la touche -
  • Pour faire un dézoom sur Linux : utiliser CTRL puis la touche -

De plus, les couleurs utilisées pour illustrer les différentes pages Web respectent les différents symptômes de daltonisme, et donc proposent des pages Web adaptées.
