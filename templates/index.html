Fragments de page

Les bouts de code HTML présentés ci-dessous peuvent servir à assembler une nouvelle page. Tous les fichiers sont encodés en UTF-8. Ils ne sont pas destinés à être visualisés dans un navigateur.

  • Boîte de contact

    Boîte d'informations de contact (exemple d'un laboratoire fictif)

  • Boîte de texte

    Boîte de texte libre

  • Boîte de liens 1

    Boîte de liens, une seule colonne

  • Boîte de liens 2

    Boîte de liens, deux colonnes

  • Boîte de citation

    Boîte contenant une citation, ainsi qu'une photo

  • Boîte couleur faculté

    Boîte de texte respectant la couleur de faculté (la couleur de faculté est définie en ajoutant la CSS spécifique à la faculté)

  • En-tête commun à toutes les pages de l'EPFL

    L'en-tête est composé de deux parties:
    1. La signature, à inclure obligatoirement dans l'entête HTML: header.sig.html.
    2. Le header traduit: version française | version anglaise

  • Pied de page

    Pied de page commun à toutes les pages de l'EPFL

  • Navigation horizontale

    Navigation horizontale située sous le titre du site. Nécessite Javascript pour fonctionner correctement (affiché sous forme de liste sinon).

  • Navigation verticale

    Navigation verticale située dans la colonne de droite. Nécessite Javascript pour fonctionner correctement (affiché sous forme de liste sinon).

Web 2010, c'est quoi?

  • Image de l'EPFL sur le web qui soit en adéquation avec la réputation de l'École: moderne, dynamique, innovative
  • Web orienté vers les intérêts des publics externes, et non comme un outil intranet: plus d'actualités
  • Professionnalisme
  • Meilleures règles d'urbanisme et de navigation dans le domaine epfl
