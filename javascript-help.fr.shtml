Activation de JavaScript

JavaScript offre un plus grand confort de navigation au travers des sites de l'EPFL. Vous trouverez ci-dessous des instructions permettant d'activer cette technologie dans les principaux navigateurs web.

Internet Explorer

  1. Sélectionnez "Outils" dans la barre de menus.
  2. Choisissez "Options Internet".
  3. Cliquez sur l'onglet "Sécurité".
  4. Cliquez sur "Personnaliser le niveau...".
  5. Faites défiler la liste pour afficher la section "Script".
  6. Sous "Active Scripting", sélectionnez "Activer" et cliquez sur OK.

Firefox

  1. Sélectionnez "Outils" dans la barre de menus.
  2. Choisissez "Options".
  3. Cliquez sur l'onglet "Contenu".
  4. Cochez la case "Activer Javascript" et cliquez sur OK.

Firefox (MacOSX)

  1. Sélectionnez "Firefox" dans la barre de menus.
  2. Choisissez "Préférences".
  3. Cliquez sur l'onglet "Contenu".
  4. Cochez la case "Activer Javascript" et cliquez sur OK.

Safari (MacOSX)

  1. Sélectionnez "Safari" dans la barre de menus.
  2. Choisissez "Préférences".
  3. Cliquez sur l'onglet "Sécurité".
  4. Cochez la case "Activer Javascript".

Safari (Windows)

  1. Sélectionnez "Modifier" dans la barre de menus.
  2. Choisissez "Préférences".
  3. Cliquez sur l'onglet "Sécurité".
  4. Cochez la case "Activer Javascript".

Opera

  1. Sélectionnez "Outils" dans la barre de menus.
  2. Choisissez "Préférences".
  3. Cliquez sur l'onglet "Avancé".
  4. Cliquez sur "Contenu" dans la liste de gauche.
  5. Cochez la case "Activer le Javascript" et cliquez sur OK.
